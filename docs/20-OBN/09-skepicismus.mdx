# Skepticismus

- **Doba**: od 4. století př. n. l.
- **Zakladatel**: Phyrrhón.
- **Název**: Ze slova skeptesthai (myšlení, pochybnost).
- Není možné se dopracovat naprosto pravdivých odpovědí na různé otázky.
- K různým věcem máme zajímavá rozdílná stanoviska.
- Člověk k životu nepotřebuje pravdivá tvrzení, vystačí si s tvrzeními pravděpodobnými.
