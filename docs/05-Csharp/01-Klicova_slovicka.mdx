### Enum
* hodnoty si určíme, vytvoření vlastního datového typu

```csharp
Operator znamenko;

enum Operator {
    scitani,
    odcitani,
    nasobeni,
    deleni
} 
```

:::tip datový typ Operator, má 4 možné hodnoty
    scitani,
    odcitani,
    nasobeni,
    deleni
:::

### Virtual
* Má vlastní tělo, jde přepsat v jiné **třídě**
```csharp
class Trida1 {
    protected virtual Metoda {
        // tělo metody
    }
}
```

```csharp
class Trida2 : Trida1 {
    override Metoda {
        // přepsaná metoda
    }
}
```

### Abstract
* Nemá tělo, tudíž je povinnost jej **přepsat**
```csharp
class Trida1 {
    protected abstact Metoda;
}
```
```csharp
class Trida2 : Trida1 {
    override Metoda {
        // Přepsaná metoda
    }
}
```
