# Pojmy

Redundance = Záloha (Serveru, Routeru)

## Prvky v síti
- Prvky v síti se dělí na
>- Aktivní: prvek, který s datovým signálem vykonává určitou aktivní činnost.
>>- router, switch, počítač
>- Pasivní: prvek, který s datovým signálem nijak nepracuje.
>>- kabel, zasuvka, rozdvojka
