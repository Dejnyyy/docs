# Protokoly 2 vrstvy (switche)
## Spanning tree protocol
Když máme v síti více switchů propojených dohromady, může vzniknout smyčka ()<br/>

**Bridge PDU** => datová jednotka, která je **jediná** povolena při pritokolu Spanning tree
- BID (Bridge ID) obsahuje 3 udaje
1. Priority = 32768 (čim menší, tím větší priorita)
2. Extended-System-ID = VLAN ID
3. MAC = mac SW

Postup nastavení Spanning   protokolu
1. ROOT BRIDGE = nastaví hlavní switch (jdou k němu všechna data)
2. ROOT PORTY = nastaví porty od ostatních switchu, které vedou k **ROOT BRIDGE SWITCHI**
FastEthernet = 19
GigabitEthernet = 4

3. Designated = porty protilehlé k **ROOT PORTŮM**
4. Alternate "Blocked" = 

Co se děje
1. Zapojení kabelu => oranžové světílko
2. Port = Blocked = Alternate - Poslouchá BPDU
3. Listnening - poslouchá + posilá BPDU = FD

##  EtherChannel
### LACP
- **active** - EtherChannel je zapnutý, rozesílá žádosti o spojení
- **passive** - Pouze **přijímá** žádosti o spojení, **nerozesílá**
### PAGP
- **desirable** - active
- **auto** - passive

**Výhody**
- Zrychlení přenosu dat
- Redundance
[Nastavení EtherChannelu](commandy#etherchannel)

## DHPC
Rozdává IP adresy hostům
DHCPv6 => rozdává IPv6

## DNS
Překládá domény na IP adresy 

## HTTP
HTTP je protokol pro komunikaci mezi klientem a serverem. Klient pošle požadavek, server odpoví. 

## FTP
FTP (File Transfer Protocol) je protokol pro přenos souborů mezi počítači. 

## SMTP
SMTP (Simple Mail Transfer Protocol) je protokol pro odesílání e-mailů. 

## SSH
SSH (Secure Shell) je zabezpečený protokol pro přístup na vzdálené počítače. 

## SNMP
SNMP (Simple Network Management Protocol) je protokol pro sledování stavu sítě. 

## TFTP
TFTP (Trivial File Transfer Protocol) je jednoduchý protokol pro přenos souborů. 

## Telnet
Telnet je protokol pro přístup na vzdálené počítače. 

## RDP
RDP (Remote Desktop Protocol) je protokol pro vzdálený přístup na počítač. 

## IPSec
IPSec je zabezpečený protokol pro přenos dat. 

## NAT
NAT (Network Address Translation) je protokol pro překlad IP adres. 

## NTP
NTP (Network Time Protocol) je protokol pro synchronizaci času. 

## RIP
RIP (Routing Information Protocol) je protokol pro propagaci směrovacích informací.
