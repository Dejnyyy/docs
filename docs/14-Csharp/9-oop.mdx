# OOP

## Názvosloví OOP
Je dobré vedět, o čem mluvíte


|Název|Definice|
|:-:|:-:|
|Třída|Definice objektu|
|Objekt|Jedna instance třídy|
|Metoda|Funkce třídy|
|Atribut|Proměnná třídy|
|Konstruktor|Metoda, která se volá při vytvoření objektu|
|Deklarace|Správný zápis|
|Inicializace|Přiřazení hodnoty k atributu|
|Instance|Vytvoření objektu pomocí třídy|

:::tip Příklad
```c
public class Class { // třída
    public int a; // atribut
    public Class() { // konstruktor
        a = 0; // inicializace atributu
    }
    public void Method() { // metoda
        Console.WriteLine(a); // použití atributu
    }
}

Class c = new Class(); // instance třídy
```
:::
