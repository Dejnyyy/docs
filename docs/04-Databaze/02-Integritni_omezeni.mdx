# Integritní omezení (IO)

## Primary key

- Unikátní hodnoty
- každá položka v tabulce má svojí vlastní PK
- Neobsahuje NULL (vždy nutné něco vložit)
- V tabulce nejvýše 1x

## Foreign Key

- Duplicitní hodnoty
- Může obsahovat NULL (nemusíme určit)
- Počet 0 – X

## not null

- Vždy nutné zadat hodnotu

## null

- Když nevím tak nic nezadám

## Auto increment

- Automatické číslování 1,2,3,4,…

## zerofill

- Neignoruje 0 na začátku čísla
