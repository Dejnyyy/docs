* Všechny příkazy jsou `case non-sensitive`, všichni to ale píšou velkýma :\)

# SELECT

* používá se na zvolení tabulky

```sql
SELECT <sloupec> FROM <tabulka>;
```

* pokud chceme zvolit všechny sloupce, použijeme `*`
```sql
SELECT * FROM <tabulka>;
```
