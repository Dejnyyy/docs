# EtherChannel
* EtherChannel je protokol, která umožňuje spojit dvě zařízení pomocí více kabelů

## Nastavení EtherChannelu u Switche
1. Zvolíme porty, které chceme spojit do jednoho
```
Switch(config)#interface range fe0/1-2
```

2. Vytvoříme skupinu
```
Switch(config-if-range)#channel-group <id_skupiny> mode active
``` 

:::warning VLANY a EtherChannel
Je žádoucí aby fungoval v režimu **trunku**
```
Switch(config-if-range)#switchport mode trunk
```
:::

:::warning Dynamický trunk a EtherChannel
Může se stát, že L3 switch se nedohodne s druhým switchem na trunk modu (L3 switch má automatický trunk/access)
```c title="Force trunk"
Switch(config-if)#switchport trunk encapsulation dot1q
```
:::
