# Dělení elektrotechniky
### 1. Analogová elektronika
- nabírá nekonečno mnoho hodnot
- jedná se o základní elektroniku (rezistory, součastky)

### 2. Diskrétní (číslicová)
- nabírá pouze určený počet hodnot
- většinou nabírá dvou hodnot (0 nebo 1)
